株式会社カヤック - 『つくっていいとも!』UX企画発表

学校のUX授業の企画で自分達でイベントを企画し、株式会社カヤックの社内イベント「つくっていいとも!」で発表を行いました。
私のチームは、鎌倉の小学校に子どもたちが楽しくごみ問題を学べる授業の仕組みをつくりました。
こちらのイベントで1位をいただきました。制作期間は、4週間です。

The students planned their event as part of the school's UX class project and presented it at Kayac Inc.'s in-house event, "Tsukutte Iitomo!"
My team created a class system for elementary schools in Kamakura, where children can learn about garbage issues in a fun way.
We won first place at this event. It took about 4 weeks to complete the production.
